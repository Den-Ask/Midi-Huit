Midi Huit est un jeu de strarége à deux joueurs basé sur le pierre-feuille-ciseaux avec des éléments de contrôle de territoire et de gestion de ressources.
Le but du jeu est d'élliminer toutes les pièces de l'adversaire.
Chaque joueur contrôle une armée composée de différentes unités, chacune ayant ses propres forces et faiblesses.

Règles

Pièces, déplacement et faiblesses

Les joueurs ont quatre types de pièces qui peuvent se déplacer de une case dans toutes les direction avec chacune des déplacement spéciaux:
  • L'Attaquant (représenté par ✦) : Il est faible contre le Défenseur et peut avancer de deux cases en diagonale
  • Le Défenseur (représenté par ■) : Il est faible contre le Tireur et peut reculer de trois cases
  • Le Tireur (représenté par ▲) : Il est faible contre l'Attaquant et peut avancer de trois cases
  • Le Comandant (représenté par ●) : Il est faible contre toutes les autres pièces et se déplace normalement
Pour attaquer une pièce adverse, il faut se placer dans la même case qu'elle.
Selon le type de chaque pièce belligérante qui se rencontrent, le résultat sera différant :
  • Si l'assaillant est plus fort, il remplace la cible
  • Si l'assaillant est équivalant, il disparait avec la victime
  • Si l'assaillant est plus faible, il disparait tous seul

La monnaie

Le jeu a une économie basée sur une monnaie appelée le Mpesa.
Il y a 10 Mpesa en circulation dans la partie.
Avant de jouer son tour, le joueur reçoit 1 Mpesa (si il en reste).
Les Mpesa sont consommés (et renvoyés à la banque) pour faires différentes actions spéciales :
  • Le joueur peut créer une unité (✦/■/▲) à côté de son commandant (si il est encore en jeu) contre 1 Mpesa si il a mois de trois unités du type qu'il veut créer
  • Le joueur peut changer le type d'une unité (ex : ▲ → ✦) contre 3 Mpesa si l'unité n'est pas menacée et qu'il a mois de trois unités du type qu'il veut créer
  • Le joueur peut créer un nouveau commandant (si il n'en a plus) sur une case libre de sa première ligne contre 5 Mpesa et un sacrifice de une unité de chaque type

Jeu

Cliquer sur une unité pour la sélectionner, puis cliquer sur une case adjacente pour la déplacer ou sur une case occupée par une unité adverse pour l'attaquer.
Il est impossible de passer par dessus une unité, même pour les déplacements spéciaux.
Chaque joueur à de une barre d'options pour créer ou modifier une pièce :
  • Côté Tireur (à gauche du joueur 1) pour la création
  • Côté Attaquant (à droite du joueur 1) pour la modification
La case d'une option est par déffaut grise : Elle deviens verte si le joueur peut utiliser l'option.
Les cases sont toutes rouges si le joueur est en mode sacrifice puis vertes si le joueur est en mode invocation de commandant.
Cliquer sur l'unité de création après avoit sellectionné le commandant puis sur une case adjacente à au commandant pour créer une nouvelle unité de ce type (si les conditions sont remplies).
Cliquer sur une unité déjà en jeu puis sur une option de modification pour changer son type (si les conditions sont remplies).
Cliquer sur l'option de création du commandant puis cliquer sur les pièces pièces qui seront sacrifiées pour créer un nouveau commandant (si les conditions sont remplies).
Cliquer sur le boutton "HH" pour réinitialiser la partie.
Cliquer sur le boutton "ℹ" pour afficher ces informations.
